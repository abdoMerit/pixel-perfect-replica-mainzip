# HopeRise — Empowering Communities

A nonprofit organization website built with React, Vite, TanStack Router, and shadcn/ui components. Originally created with [Lovable](https://lovable.dev).

## Stack

- **Framework**: React 19 + TanStack Router (file-based routing)
- **Build tool**: Vite 8 via `@lovable.dev/vite-tanstack-config`
- **Styling**: Tailwind CSS v4 + shadcn/ui component library
- **Runtime / package manager**: Bun

## Pages

- `/` — Home
- `/about` — About
- `/programs` — Programs index + `/programs/:slug` — individual program
- `/projects` — Projects
- `/news` — News
- `/events` — Events
- `/donate` — Donate
- `/volunteer` — Volunteer
- `/contact` — Contact

## Running locally on Replit

```bash
bun run dev     # starts dev server on port 5000
bun run build   # production build
bun run lint    # ESLint
```

The workflow **"Start application"** (`bun run dev`) is already configured and will auto-start.
